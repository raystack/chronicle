# API Overview

Chronicle provides a simple API for building documentation.

## Core Concepts

- **Pages** - MDX files that become documentation pages
- **Config** - YAML configuration file
- **Themes** - Customizable UI themes

## File Structure

```
my-docs/
├── chronicle.yaml
├── index.mdx
└── guides/
    └── getting-started.mdx
```
