# Installation

## System Requirements

| Requirement | Version |
|-------------|---------|
| Node.js     | 18+     |
| npm/pnpm    | Latest  |

## Install via npm

```bash
npm install @raystack/chronicle
```

## Install via pnpm

```bash
pnpm add @raystack/chronicle
```

## Verify Installation

```bash
npx chronicle --version
```
